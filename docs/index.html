Boitata - HeatSentinel

Desafio: Desenvolver uma palicação completa, com interface de sistema, interface de usuário e banco de dados normalizado, em modelo ao BDQueimadas.

Realizado em grupo, junto a outros três colegas de turma, desenvolvemos uma aplicação web que busque em banco de dados próprio, e mostra ao usuário informações climaticas específicas, como focos de calor, áreas queimadas e risco de fogo, com opões de filtragens por area ou termporal.

Desenvolvido em tecnologia HTML, CSS, JavaScript, sem a utilização de frameworks ou afins.
